Modul Pembelajaran Kami

Koding Next adalah tempat belajar coding yang seru untuk anak-anak, sekaligus investasi skill yang mendukung pendidikan dan masa depan mereka.


Little Koders (Usia 4 – 8 tahun)


Mengenalkan dasar-dasar coding secara menyenangkan untuk melatih logika dan kreativitas sejak dini.

Our Modules
for Little Koders

Coding Stories

Usia 4 - 6 (45 menit / sesi)

Tentang Kursus:

Belajar coding sambil bermain!

Mengenal logika pemrograman dasar dengan membuat animasi, bermain board game interaktif, dan menulis kode sederhana lewat cerita yang menarik. Cocok untuk membangun cara berpikir logis sejak dini, dengan metode yang menyenangkan dan tidak membosankan!

Games & Apps

Usia 6 - 8 (60 menit / sesi)

Tentang Kursus:

Bikin game dan animasi dari tablet!

Belajar menyusun perintah sambil membuat game interaktif dan animasi sederhana. Kursus ini juga melatih mereka berpikir sistematis dan bekerja sama (multithreading). Semua kegiatan dilakukan lewat tablet yang mereka sukai!

Coding with Math

Usia 6 - 8 (60 menit / sesi)

Tentang Kursus:

Anak-anak akan dilatih berpikir logis lewat tantangan matematika dalam permainan. Mereka akan mempraktikkan bentuk-bentuk dasar, perkalian, logika perbandingan, dan keterampilan matematika lainnya. Semua ini akan mereka gunakan untuk membuat aplikasi, game 3D sederhana, dan memecahkan teka-teki.

Computer Programming

Usia 6 - 8 (60 menit / sesi)

Tentang Kursus:

Belajar coding pakai komputer, bukan tablet!

Di kelas ini, anak-anak akan membuat game dan proyek interaktif sambil belajar dasar penggunaan komputer—seperti klik mouse dan mengetik. Kelas ini jadi jembatan ke level coding lanjutan, sekaligus melatih fokus, logika, dan kemandirian sejak awal.




Junior Koders (Usia 8 - 16 tahun)


Program lanjutan yang mencakup pemrograman visual seperti pembuatan Game 2D dan Aplikasi Smartphone, hingga pemrograman berbasis teks seperti Python, JavaScript, dan proyek teknologi Smart Home IoT.

Our Modules
for Junior Koders

2D Games Development with Roblox

Usia 8 - 12 (90 menit / sesi)

Tentang Kursus:

Bikin game sendiri? Bisa dong!

Anak-anak akan belajar animasi dan pemrograman game 2D melalui proyek-proyek seru. Mereka diajak memahami elemen dasar game, lalu menerapkannya untuk menciptakan game buatan mereka sendiri di Roblox Studio.

Mobile Apps Development

Usia 8 - 12 (90 menit / sesi)

Tentang Kursus:

Mulai bikin aplikasi dari 0!

Anak-anak akan belajar cara membangun halaman web sederhana menggunakan HTML dan CSS. Cocok untuk mengenal logika desain web dan struktur aplikasi digital di dunia nyata.

Roblox

Usia 8 - 12 (90 menit / sesi)

Tentang Kursus:

Bikin games Roblox versimu sendiri!

Belajar bahasa pemrograman LUA dan cara menggunakannya di Roblox Studio. Mereka akan memulai dari dasar, lalu melanjutkan hingga bisa membuat game Roblox versi mereka sendiri dengan desain dan logika yang mereka bangun sendiri.

Website Development

Usia 8 - 16 (90 menit / sesi)

Tentang Kursus:

Bangun website pertama mu sendiri!

Anak-anak akan belajar membuat dan mengatur tampilan situs web mereka sendiri. Mereka akan menulis kode menggunakan HTML dan CSS, serta memahami struktur situs dan bagaimana halaman web terhubung dan bekerja secara keseluruhan.

Python

Usia 12 - 16 (90 menit / sesi)

Tentang Kursus:

Mulai belajar coding pakai teks!

Anak-anak akan diajak mengenal konsep pemrograman seperti variabel, fungsi, dan logika if-else menggunakan Python. Mereka juga akan membuat program dengan tampilan visual sendiri (GUI)!

Javascript

Usia 12 - 16 (90 menit / sesi)

Tentang Kursus:

Bikin situs web yang interaktif? Bisa banget!

Anak-anak akan belajar bahasa JavaScript yang dipakai di seluruh dunia. Mulai dari dasar sampai bisa membuat halaman web yang hidup, lengkap dengan HTML sebagai pendukungnya.

Internet of Things (IoT)

Usia 12 - 16 (90 menit / sesi)

Tentang Kursus:

Perpaduan coding dan hardware!

Anak-anak akan merakit perangkat pintar dan memprogramnya menggunakan bahasa C/C++. Mereka akan membuat proyek yang bisa digunakan di rumah, sambil memahami bagaimana dunia digital dan benda nyata bisa terhubung!

GRATIS! Kelas Pertama

Tentang Kami

Sebagai bagian dari franchise Koding Next yang berkembang pesat di Indonesia, Koding Next East Java berkomitmen untuk menjadi tempat bertumbuhnya calon-calon inovator muda. Kami percaya bahwa setiap anak memiliki potensi untuk menjadi pencipta teknologi masa depan. Mari melangkah bersama—belajar, berkreasi, dan menjadi bagian dari masa depan teknologi bersama Koding Next East Java.

2022 - Hadir di Surabaya

Cabang pertama resmi dibuka di Keputih, Sukolilo. Langkah awal Koding Next di Jawa Timur!

2023 - Kolaborasi dengan Sekolah

Menjalin kemitraan dengan sekolah-sekolah, menghadirkan pembelajaran coding untuk jenjang TK hingga SMA.

2024 - Memperluas Ekspansi & Integrasi Kurikulum

Memperluas kerja sama dan mendukung sekolah dalam mengintegrasikan coding sebagai bagian dari kurikulum wajib.
