Modul Pembelajaran Kami

Koding Next menawarkan kesempatan yang unik dan menarik bagi anak-anak untuk belajar tentang dunia coding dan teknologi dengan cara yang menyenangkan.



Little Koders (Usia 4–8 tahun)

Mengenalkan dasar-dasar coding secara menyenangkan untuk melatih logika dan kreativitas sejak dini.

Our Modules
for Little Koders

Coding Stories

45 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar dasar coding sambil bermain. Mereka akan dilatih untuk berpikir logis lewat aktivitas seru dan alat yang sesuai dengan usia mereka. Kegiatannya termasuk membuat animasi, bermain board game yang punya alur cerita, dan mencoba menulis kode sederhana.

Games & Apps

60 menit / sesi • 20 sesi



Tentang Kursus:

Melalui iPad atau tablet, anak-anak akan dikenalkan cara memecahkan masalah. Mereka akan belajar membuat animasi dan game sederhana menggunakan konsep seperti urutan perintah, pengulangan (loop), kejadian (event), hingga bekerja secara bersamaan (multithreading).

Coding with Math

60 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan dilatih berpikir logis lewat tantangan matematika dalam permainan. Mereka akan mempraktikkan bentuk-bentuk dasar, perkalian, logika perbandingan, dan keterampilan matematika lainnya. Semua ini akan mereka gunakan untuk membuat aplikasi, game 3D sederhana, dan memecahkan teka-teki.

Computer Programming

60 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar membuat game dan proyek interaktif menggunakan komputer, bukan tablet. Mereka juga akan mulai mengenal cara menggunakan mouse dan mengetik, sebagai persiapan menuju level coding yang lebih lanjut.




Junior Koders (Usia 8-12 tahun)

Program lanjutan yang mencakup pemrograman visual seperti pembuatan Game 2D dan Aplikasi Seluler, hingga pemrograman berbasis teks seperti Python, JavaScript, dan proyek teknologi Smart Home IoT.

Our Modules
for Junior Koders

2D Games Development with Roblox

90 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar membuat dan memprogram animasi serta game sederhana melalui berbagai tugas dan proyek. Mereka akan dikenalkan pada elemen dan konsep dasar dalam pembuatan game, lalu menerapkan pengetahuan tersebut untuk menciptakan game buatan mereka sendiri.

Mobile Apps Development

90 menit / sesi • 20 sesi



\

Tentang Kursus:

Anak-anak akan mempelajari dasar-dasar pemrograman situs web menggunakan HTML dan CSS. Mereka akan belajar membuat halaman web statis yang sederhana, sekaligus memahami bagaimana struktur dan susunan sebuah situs web dibangun.

Roblox

90 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar bahasa pemrograman LUA dan cara menggunakannya di Roblox Studio. Mereka akan memulai dari dasar, lalu melanjutkan hingga bisa membuat game Roblox versi mereka sendiri dengan desain dan logika yang mereka bangun sendiri.

Website Development

90 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar membuat dan mengatur tampilan situs web mereka sendiri. Mereka akan menulis kode menggunakan HTML dan CSS, serta memahami struktur situs dan bagaimana halaman-halaman web terhubung dan bekerja secara keseluruhan.

Python

90 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan diperkenalkan dengan pemrograman berbasis teks, dan mempelajari konsep dasar seperti variabel, fungsi, pernyataan bersyarat (seperti if-else), serta cara menggunakan pustaka kode (library). Mereka akan menggunakan semua ini untuk membuat program dengan tampilan visual (GUI).

Javascript

90 menit / sesi • 20 sesi



Tentang Kursus:

Anak-anak akan belajar membuat kode menggunakan JavaScript, salah satu bahasa pemrograman paling populer di dunia. Mereka akan mulai dari dasar, memahami cara kerja JavaScript di dalam situs web, dan juga diperkenalkan dengan HTML untuk membangun halaman interaktif.

Internet of Things (IoT)

90 menit / sesi • 20 sesi



Tentang Kursus:

Dalam kelas Internet of Things (IoT), anak-anak akan mempelajari konsep dasar perangkat pintar yang terhubung ke internet. Mereka akan membuat proyek-proyek sederhana yang berguna untuk kehidupan sehari-hari di rumah. Selain itu, mereka akan belajar cara merakit dan memprogram mikrokontroler menggunakan bahasa pemrograman C atau C++.

Tentang Kami

Sebagai bagian dari franchise Koding Next yang berkembang pesat di Indonesia, Koding Next East Java berkomitmen untuk menjadi tempat bertumbuhnya calon-calon inovator muda. Kami percaya bahwa setiap anak memiliki potensi untuk menjadi pencipta teknologi masa depan. Mari melangkah bersama—belajar, berkreasi, dan menjadi bagian dari masa depan teknologi bersama Koding Next East Java.
